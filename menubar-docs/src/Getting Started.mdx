# Getting Started

## Installation

Start by installing `cb-react-menubar` as a dependency with `npm` :

```
npm i cb-react-menubar
```

or if you are using `yarn` :

```
yarn add cb-react-menubar
```

## Basic Usage


### 1.Define the data

* Define the data(menu items) which you wanna render in the Menubar

* If you want back in a menubar , give value as back . You will provided with back option

* If you dont want nested items for a specific item , Give it in items array

* If you dont want nested items for a specific item , dont declare the items array or give a empty array for the items

```
 const menuItems = {
      value: 'menu-items',
      items: [
        {value: 'Fashion',
          items: [
            {value: 'back'},
            {value: 'Men',
              items: [
                {value: 'back'},
                {value: 'Shirts'}
              ]
            },
        },
        {value: 'Electronics',items: []},
        {value: 'Furnitures',items: []},
        {value: 'Jewelery&watches',items: []}
      ]
    }

  export default MenuItems;
```


### 2. Use the Menubar component anywhere in your code

Use the Menubar component and pass in the `data`.

```
import MenuBar from 'cb-react-menubar';
import MenuItems from "../path/where/menuItems/present"

const app=()=>{
  <div>
    <h1>This is Animated Menu-bar</h1>
    <MenuBar data={MenuItems}/>
  </div>
}
```

### 3. Styling the Menubar

* You can style the Menubar by passing backgroundColor and textColor

* You can even have your own className and pass to the Menubar

```
import MenuBar from 'cb-react-menubar';
import MenuItems from "../path/where/menuItems/present";
import "./index.css"

const app=()=>{
  <div>
    <h1>This is Animated Menu-bar</h1>
    <MenuBar backgroundColor="blue" textColor="white" style="menuBar" data={MenuItems}/>
  </div>
}

```


### 4.Animations

* You are provided with various animation effects along with menubar

* It should be an array which denotes the entering and exiting of the menuItems in the menuBar

```
import MenuBar from 'cb-react-menubar';
import MenuItems from "../path/where/menuItems/present";
import "./index.css"

const app=()=>{

  const animationNames=['slideIn','slideOut']

  <div>
    <h1>This is Animated Menu-bar</h1>
    <MenuBar animation={animationNames} data={MenuItems}/>
  </div>
}

```
